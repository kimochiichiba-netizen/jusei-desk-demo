# JUSEI DESK — 接骨院 事務アシスタント（デモ）

接骨院・整骨院の「予約 〜 受付 〜 施術録 〜 料金計算 〜 月末のレセプト準備」の
事務仕事を下書き・点検・自動化するツールのデモ画面です。

## 3つのデモ

| ページ | 内容 |
|---|---|
| [index.html](index.html) | **本体デモ**。受付・問診 → 保険適用の判定 → 負傷原因文の下書き（返戻チェック付き）→ 施術録12項目 → 2026年7月改定対応の料金計算 → 月末レセプトボード → 本部ビュー（10院集計）→ 返戻・照会・交通事故の回答文。**画面左上のスイッチャーでUIを7パターンに切替可能** |
| [yoyaku.html](yoyaku.html) | **予約ボード**。スタッフ×時間の担当割りボードに、LINE予約が電話対応ゼロで自動配置される実演＋自費限定クーポンの自動配布設計＋既存予約システムとの連携方式の比較 |
| [line.html](line.html) | **らくらく受付**。患者さまがトーク画面（メッセージアプリ連携のイメージ）で答えるだけで、院側に判定・負傷原因文の下書き・受付メモが届く様子の実演 |

## 注意

- 施設・患者・データはすべて架空です。入力はブラウザ内で完結し、外部へ送信されません。
- 文章・判定・料金はすべて「下書き・参考」であり、施術録・申請書の確定は施術者が行う設計です。
- 療養費の単価は2026年（令和8年）7月1日施行の改定値。判定しきい値はデモ用の代表値です。
- LINEはLINEヤフー株式会社の商標です。本デモの トーク画面は連携イメージであり、同社のサービスではありません。
